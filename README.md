# 🥕 Vegetable Management System (Python + MySQL)

## 📌 Project Overview
This is a **console-based vegetable shop management system** built using **Python and MySQL**.

The system simulates real-world shop operations like inventory management, cart handling, billing, profit calculation, and customer tracking.

---

## 🚀 Features

### 👨‍🌾 Owner (Admin)
- Add new vegetables
- Update vegetable details (cost, quantity, price)
- Delete vegetables
- View inventory
- View customer details
- View profit reports
- Generate daily reports

---

### 🛒 User (Customer)
- Add vegetables to cart
- Remove vegetables from cart
- Modify cart items
- View cart details
- Billing system with total calculation

---

### 📦 Inventory Management
- Real-time stock updates
- Prevents ordering more than available quantity
- Automatically reduces stock after purchase

---

### 💰 Profit Calculation
- Calculates profit per item
- Displays total revenue
- Tracks item-wise profit

---

### 📊 Reporting System
- Shows:
  - Sold items
  - Profit summary
  - Remaining stock

---

### 🧾 Customer Management
- Stores customer name and phone number
- Validates phone number input
- Maintains purchase history

---

## 🏗️ Tech Stack

- Language: Python
- Database: MySQL
- Library: mysql-connector-python
- Interface: CLI (Command Line Interface)
